# 📅 일정 맞추기

친구들과 모임 날짜를 쉽게 조율할 수 있는 웹 서비스예요.  
방을 만들고 링크를 공유하면, 각자 가능한 날짜를 선택할 수 있어요. 실시간으로 누가 언제 되는지 한눈에 볼 수 있어요.

🔗 **배포 링크**: https://313dpdms-code.github.io/scheduling-/

---

## 주요 기능

- **방 만들기** — 제목과 날짜 범위를 설정해서 방을 생성해요
- **링크 공유** — 방 코드 또는 링크를 친구에게 공유해요
- **날짜 선택** — 달력에서 클릭 또는 드래그로 가능한 날짜를 표시해요
- **실시간 동기화** — 참여자들의 일정이 실시간으로 반영돼요
- **겹치는 날짜 확인** — 모두가 가능한 날짜를 자동으로 알려줘요
- **방 제목 수정** — 방장이 언제든 제목을 바꿀 수 있어요
- **방 삭제** — 방을 만든 사람만 방을 삭제할 수 있어요

---

## 사용 방법

### 방 만들기
1. 방 제목과 날짜 범위를 입력하고 **방 만들기** 클릭
2. 이름과 색상을 선택하고 **참여하기** 클릭
3. 달력에서 가능한 날짜를 클릭 또는 드래그로 표시
4. 하단 또는 상단의 **링크 복사** 버튼으로 친구에게 공유

### 링크로 참여하기
1. 공유받은 링크 또는 방 코드를 입력하고 **일정 보기** 클릭
2. 이름과 색상을 선택하고 **참여하기** 클릭
3. 달력에서 가능한 날짜 선택

---

## 달력 색상 표시

| 색상 | 의미 |
|------|------|
| 🟢 초록 배경 | 모든 참여자가 가능한 날 |
| 🟡 노랑 배경 | 절반 이상 가능한 날 |
| 🔴 연빨강 배경 | 일부만 가능한 날 |

## 로컬 실행 방법

Firebase를 사용하기 때문에 `file://` 방식이 아닌 로컬 서버로 실행해야 해요.

```bash
# Python이 설치되어 있는 경우
python3 -m http.server 8080
```

브라우저에서 `http://localhost:8080` 접속
